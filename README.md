## Cur8's Team Wiki

-------
**STEPS TO RUN THIS PROJECT**:

start frontend (client): 
`yarn run dev`

start backend (server) in another terminal window: 
`flask --app vids run`
or 
`python3 -m flask --app vids run`

*client is being held at http://localhost:3000. server is http://localhost:5000
